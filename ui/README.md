# Nexus UI
This source code initially generated by Claude but using javascript, I migrated it to typescript later!

React + Vite + TypeScript+ Tailwind frontend for the Nexus local AI agent, structured
after Unsloth Studio's layout: collapsible sidebar with nav + recents,
top model selector, and a chat area with paste/drag-drop image support
and Think / Search / Code toggles.

## Setup

```bash
cd nexus-ui
npm install
npm run dev
```

Runs at `http://localhost:5173`.

## Wiring to your FastAPI backend

Everything currently talks to placeholder routes on `127.0.0.1:8000`.
Your `main.py` currently only exposes `/`, `/health`, and `/test`.

To connect the chat UI for real:

1. Add a real chat route in `main.py`, e.g.:
   ```python
   @app.post("/chat")
   async def chat(payload: ChatRequest):
       result = app.state.workflow.invoke({"messages": payload.messages})
       return {"content": result["messages"][-1].content}
   ```
2. Update `src/api/client.ts` → `sendMessage()` to match your request/response
   shape.
3. If you want live model list / conversation persistence, implement
   `/models` and `/conversations` and swap out `MOCK_MODELS` and the
   in-memory `conversations` state in `App.jsx` for data fetched via
   `api.listModels()` / `api.listConversations()`.
4. CORS: since the UI runs on `localhost:5173` and the API on
   `127.0.0.1:8000`, add `CORSMiddleware` to `main.py`:
   ```python
   from fastapi.middleware.cors import CORSMiddleware
   app.add_middleware(
       CORSMiddleware,
       allow_origins=["http://localhost:5173"],
       allow_methods=["*"],
       allow_headers=["*"],
   )
   ```
